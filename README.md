#  Broader Named Entity Identification and Linking

Ontology classes recognized by script are:
- dbo:Building
- dbo:Animal
- dbo:Person
- dbo:University
- dbo:Organisation

When no classes are avaliable for a token on dbpedia, the class "http://localhost/Other" is assigned.
